Classes summary

CSVMaker

This class will be used to make CSV files. A little less useful/required than the Parser tool, but helps keep a standard for writing them
CSVParser

Provides Iterator access to a CSV file. Includes options to use the files headers to create JSON objects, allows for headings to be swapped and the use of dot notation in headings to structure the data internally. Any filtering/typecasting of results will be the responsibility of the end user in order to keep this class minimal
